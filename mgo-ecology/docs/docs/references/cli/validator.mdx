MGO CLI `validator` provides command-line access to the validator functions of the Mango network.

## Commands

The following list details all the available sub-commands for the MGO move command.

```bash
Validator and validator candidate tools

Usage: mgo.exe validator [options] [command]

Commands:
  make-validator-info                Create validator information
  become-candidate                   Become a candidate
  join-committee                     Join the committee
  leave-committee                    Leave the committee
  display-metadata                   Display metadata
  update-metadata                    Update metadata
  update-gas-price                   Update the gas price used to calculate the reference gas price
  report-validator                   Report or unreport a validator
  serialize-payload-pop              Serialize payload for generating proof of possession. Useful for offline processing of a payload for Authority protocol key pair signing
  display-gas-price-update-raw-txn   Print out the serialized data of a transaction setting the validator's gas price quote
  help                               Print this message or the help information of a given sub-command

Options:
      --client.config <CONFIG>   Set the file that stores our user account state (if missing, an empty file will be created)
      --json                     Return command output in JSON format
  -y, --yes
  -h, --help                     Print help
  -V, --version                  Print version
```

## JSON Output

Append the --json flag to the command to format the response in JSON, instead of the more human-friendly default MGO CLI output. This can be useful for large datasets, which may display issues on smaller screens.
