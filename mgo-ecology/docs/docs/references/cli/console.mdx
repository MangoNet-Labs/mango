---
title: Console
---

The MGO CLI `console` command provides command-level access to interact with the Mango network by wrapping MGO client CLI commands in a bash-like functionality. This command starts a new process and provides users with an environment to run all available MGO Client CLI commands. Additionally, it offers command history support.

## Commands

The following list enumerates all available subcommands of the MGO console command.

```bash
Start the Mgo interactive console

Usage: mgo.exe console [options]

Options:
      --client.config <CONFIG>  Set file to store our user account state (if missing, creates an empty file)
  -h, --help                    Print help
  -V, --version                 Print version

```
